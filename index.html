CARA

Chess Analysis and Review Application

Free & open source • Windows macOS Linux

Version 2.7.5

A full-featured desktop application for analyzing and reviewing chess games. CARA helps you understand your games, identify mistakes, recognize patterns, and improve your chess understanding.

A few highlights

Some distinctive features of CARA are:

Multi UCI Engine support
Use any UCI-compatible engine(s) and assign them per task—evaluation, game analysis, manual analysis, and brilliancy detection.
Learn more...
Extensive game summary, highlight rules, and player statistics
Get a condensed overview with evaluation graph, key moments, detected highlights and per-player performance and trends.
Learn more...
Manual analysis with piece trajectory exploration
Visualize the engine’s plan by tracing piece paths from principal variations.
Learn more...
AI summary (OpenAI, Anthropic, custom endpoints)
Discuss positions with an LLM that can reference engine analysis and game context—pick your provider.
Learn more...
Tagging + auto-tagging
Organize your games with colored tags, including Auto-Tagging based on analysis results.
Learn more...
Moves List “data workbench”
A comprehensive, customizable moves table centralizes all per-move information in one place.
Learn more...
More: Highly customizable • Online import (Chess.com/Lichess) • Manage multiple databases • Theme support • Bulk analysis • Bulk PGN header management • Deduplication • Powerful search • Move classification icons • Annotations • Notes
